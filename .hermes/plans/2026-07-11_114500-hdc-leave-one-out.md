# HDC Leave-One-Out 全市場比較 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 讓所有 The Odds API spreads 賽事都可用 leave-one-out 共識選出最佳 bookmaker／方向；HKJC 只在同場同盤時標記及參與候選。

**Architecture:** 改動現有 `buildHandicapCards`，以外圍賽事作分組 owner，逐 bookmaker 排除自身後計公平概率及 edge。沿用現有 parser、team matcher、Dashboard cards、snapshot 及 settlement，不加 dependency。

**Tech Stack:** TypeScript、React、Vitest、Node.js。

## Global Constraints

- 只做 HDC。
- 同場不同 line 不混合。
- 至少兩間獨立 bookmaker 才分析。
- `edge >= settings.edgeThreshold` 才買。
- Snapshot 保存實際選中的 bookmaker。
- HKJC 不再是必要條件。
